# page-a8-resource

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态资源形式存储，方便直接访问或嵌入其他项目中使用。仓库本身不针对任何特定域名或网站，仅作为 HTML 页面的集中存放与版本管理场所。

## 目录结构

```
page-a8-resource/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面所需的样式、脚本、图片等附属资源
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，文件名即页面标识。
- `assets/`：按需组织子目录，存放页面引用的外部资源。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 每个页面应自包含其必要资源，或通过相对路径引用 `assets/` 下的文件。
- 页面内容遵循通用 Web 标准，兼容主流浏览器。
- 归档页面可能不定期更新或移除，建议以仓库当前状态为准。

## 使用方式

1. 直接访问 `pages/` 下的 HTML 文件（通过 GitHub Pages 或本地预览）。
2. 克隆仓库到本地，用浏览器打开相应页面文件。
3. 如需引用页面资源，请使用仓库内相对路径。
